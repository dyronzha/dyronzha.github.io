XR防盲小尖兵

  1. 作品集_專案
  2. XR防盲小尖兵
  3. 2018/07 ~ 2018/12

大四暑假參加科技部的計劃進到資策會實習半年並接手一項來自台灣防盲基金會的案子，協助他們進行衛教計畫。 APP內容分為兩部分，首先是透過相機擷取現實影像再利用後處理來模擬各種眼睛疾病的狀況，以VR呈現讓使用者能身歷其境的體驗各種眼疾；另外則是小遊戲部分，可以讓使用者學習正確用眼知識， 也有找物品遊戲讓使用者體驗正常視力與近視的差別，最後則是在3D場景中模擬患有眼疾人士的一天來實際體驗他們的不便。

獲獎紀錄
  • 2018跨域數位人才加速躍升計畫DIGI+數位新星大賞第三名

學習重點

  • Image Effect
    • 為了達到一些眼疾效果，需要將相機所得到的畫面做圖片效果的後處理，像是近視遠視的模糊、黃斑部病變的扭曲等等
    • 在近視遠視中有特別為了讓物品會隨著遠近模糊程度有所不同，學習關於高斯模糊的原理
  • Cardboard VR
    • 使用 google 提供的插件能方便使遊戲變成 VR 模式,使用他們提供的攝影機腳本和注視事件腳本
    • 使用 VR 須注意物品不能離攝影機太近，不然會有無法對焦的問題,另外 UI 需換成在世界座標中，不能使用 overlap 的方式
    • 為了使使用者有良好體驗,有特別做 UI 介面跟隨使用者視線,抓取攝影機旋轉角度來移動旋轉 UI 物件
  • Vuforia
    • vuforia 是一個 AR 插件，使用上滿簡單的，不過為了達到一些功能,仍需要上官網查看 API，像是開關相機或調整相機對焦
  • 360 圖片
    • 為了達到業主要求有使用 360 度相機拍攝真實畫面放入 App 中
    • 360 圖片裡的物品選取邊框是偷吃步，讓美術個別生出邊框直接放入場景中對應位置
  • 其他
    • 第一次直接與業界接觸，企劃、美術和程式都有詳細的分工,自己也從中學習到很多，如規格書撰寫和如何向美術開需求，與設計彈性的code能不斷修改以便達到業主的要求
